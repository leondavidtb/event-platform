## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/leondavidtb/event-platform.git
```

Entre no diretório do projeto

```bash
  cd event-platform
```

Instale as dependências

```bash
  yarn install
```

Inicie o servidor

```bash
  yarn dev
```
